Для работы программы необходима ОС linux, а так же:

1. Установить интерпретатор Ruby версии не менее 2.0.0
2. Установить RubyGems (установщик внешних библиотек - "гемов")
3. Установить гем opengl-bindings (набрав в консоли "gem install opengl-bindings")
4. Найти библиотеки libGL.so и libglut.so и сделать на них символьные ссылки из каталога /usr/lib (либо заменить пути в 7 и 8 строках файла engine.rb).

Запускать файл engine.rb командой "ruby engine.rb"; либо "./engine.rb", предварительно установив права на запуск ("chmod +x engine.rb")
